Canvas demo

Tutorial

The html canvas is used to draw graphics onto a page using the canvas element.
The canvas element only contains the graphics, to actually draw them javascript is needed.
Using canvas many things can be drawn, in this specific example,
I am demonstrating how to draw and color a line, circle and rectangle.
The cavas here is 500 x 500, when putting in values keep this in mind.
Stroke is the outline of the shape and fill is the color the inside.

Html Canvas can be used for various different multimedia systems,
For example it can be used to draw graphs, combine photos or create animations.

Line

To draw a line 2 commands are needed, "context.moveto" which brings the coordiantes
beginning position and "context.lineto" which draws a line to the coordiantes.
4 parameters are required which are coordiantes 2 x and 2 y.
The first x and y coordinates are where the line begins and the last two are where they finish.
Line Tutorial Website

Circle

To draw a circle the arc() method is called with 5 parameters.
The first 2 parameters are the x and y coordinates represent the centre of the circle.
The third parameter is radius length. The last two are start angle and end angle
which are both in radians. The circle will start at the start angle and end at the end angle.
Circle Tutorial Website

Rectangle

To draw a rectangle the method strokeRect() is used.
The rectangle 4 parameters are required, the first two are the x and y coordiante of where
the rectangle will start. The next 2 are the width and the height of the rectangle.
Rectangle Tutorial Website
